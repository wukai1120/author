# ✍️ Author — منصة الكتابة الإبداعية المدعومة بالذكاء الاصطناعي

> استوديو كتابة مدعوم بالذكاء الاصطناعي للروائيين وكتاب السيناريو ورواة القصص.

**Author** هي أداة للكتابة الإبداعية بمساعدة الذكاء الاصطناعي. تجمع بين محرر نصوص احترافي، ومساعد كتابة ذكي، ونظام كامل لإدارة بناء العوالم والقصص - كل ذلك في تجربة واحدة سلسة.

🌐 **العرض التجريبي**: [author-delta.vercel.app](https://author-delta.vercel.app)

---

## ✨ الميزات الأساسية

### 📝 محرر احترافي
- محرر نصوص غني مدعوم بـ **Tiptap**.
- دعم المعادلات الرياضية **KaTeX** بدون إنترنت.
- خطوط وألوان وأحجام نصوص قابلة للتخصيص.
- دعم التصدير إلى Markdown و TXT و DOCX و EPUB و PDF.

### 🤖 مساعد الكتابة الذكي
- **دعم نماذج متعددة**: ZhipuAI / DeepSeek / OpenAI / Google Gemini / Claude وغيرها.
- **محرك السياق**: يقوم الذكاء الاصطناعي تلقائيًا بقراءة ملفات تعريف الشخصيات، وبناء العالم، والفصول السابقة للحفاظ على اتساق القصة.
- الإنشاء وإعادة الكتابة في الوقت الفعلي مع معاينة النص (Ghost Text).

### 📚 مدير بناء العوالم
- إدارة هيكلية (شجرة) للشخصيات والمواقع والعناصر والخطوط العريضة.
- يتم حقن الإعدادات تلقائيًا في ذاكرة الذكاء الاصطناعي.

### 💾 إدارة البيانات (محلياً وسحابياً)
- **التخزين المحلي**: يتم تخزين 100% من البيانات في متصفحك (IndexedDB).
- **المزامنة السحابية**: دعم المزامنة السحابية عبر الأجهزة باستخدام Firebase.
