# ✍️ Author — Платформа для писателей на базе ИИ

> Писательская студия на базе ИИ для романистов, сценаристов и авторов историй.

**Author** — это инструмент для творчества с поддержкой ИИ, предназначенный для писателей-фантастов и авторов романов. Он объединяет профессиональный текстовый редактор, интеллектуального ИИ-помощника и полноценную систему управления лором и сюжетом в единое целое.

🌐 **Онлайн Демо**: [author-delta.vercel.app](https://author-delta.vercel.app)

---

## ✨ Основные возможности

### 📝 Профессиональный редактор
- Многофункциональный текстовый редактор на базе **Tiptap**
- Поддержка математических формул **KaTeX** в автономном режиме
- Настраиваемые шрифты, размер текста и темы (тёмная/светлая)
- Поддержка экспорта в Markdown, TXT, DOCX, EPUB и PDF

### 🤖 ИИ-помощник писателя
- **Мультимодельная поддержка**: ZhipuAI / DeepSeek / OpenAI / Google Gemini / Claude и другие.
- **Интеллектуальный контекст**: ИИ автоматически читает профили персонажей, настройки мира и предыдущие главы для сохранения логики сюжета.
- Генерация текста в реальном времени с функцией предпросмотра (Ghost Text).

### 📚 Управление миром и сюжетом
- **Древовидная структура** для управления персонажами, локациями, предметами и планами.
- Настройки автоматически передаются в контекст ИИ.

### 💾 Управление данными (Локально и в Облаке)
- **Локальное хранение**: 100% данных хранится в IndexedDB вашего браузера.
- **Облачная синхронизация**: Поддержка мультиплатформенной синхронизации через Firebase.
